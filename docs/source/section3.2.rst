3.2 - All Information Actions
=============================

For convenience, all update configuration for a dataset can be managed
together using two comprehensive actions:

.. list-table::
   :widths: auto
   :header-rows: 0

   * - :ref:`Get All Information <section3.2.1>`
     - retrieve all update control, local file, and remote file records for a dataset
   * - :ref:`Set All Information <section3.2.2>`
     - create or modify all update control, local file, and remote file records for a dataset
