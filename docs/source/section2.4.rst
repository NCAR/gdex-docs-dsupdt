2.4 - Getting Help
=================================

| **dsupdt** [Option] -(h|help) [Option]

The description of a single option is shown when -h (-help) is given with
that option, placed either before or after -h. With no option, or when
**dsupdt** is run without arguments, the full document is displayed through
the UNIX 'more' utility. A printable copy of this document is the file
dsupdt.usg installed under the python package rda_python_dsupdt.
